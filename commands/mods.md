{% tab title="/ban" %}
## Description

> Ban a user

## Usage

Banning someone because he/she do something is not allowed on your server,

Use the command `/ban {user} {reason}`

{% hint style="info" %}
Go to unban section for see more info for unban
{% endhint %}

<figure><img src="../.gitbook/assets/image (4).png" alt=""><figcaption></figcaption></figure>

{% hint style="warning" %}
If you try to ban someone with admin permission or ban members, the bot will deny the ban
{% endhint %}

{% hint style="info" %}
You need the permission `BAN_MEMBERS` to use this command.
{% endhint %}
{% endtab %}
